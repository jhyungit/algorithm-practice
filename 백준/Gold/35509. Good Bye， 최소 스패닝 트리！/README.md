### 문제 설명

<blockquote><a href="/problem/1197">https://www.acmicpc.net/problem/1197</a></blockquote>

<p>$N$개의 정점과 $M$개의 간선으로 이루어진 가중치 있는 무방향 그래프가 주어진다. 정점은 $1$번부터 $N$번까지 번호가 붙어 있고, 간선은 $1$번부터 $M$번까지 번호가 붙어 있다.</p>

<p>그래프의 최소 스패닝 트리(MST)는 다음과 같이 정의된다.</p>

<ul>
<li>주어진 그래프의 모든 정점을 연결하는 부분 그래프 중에서 간선 가중치의 <strong>합</strong>이 최소인 트리</li>
</ul>

<p>그래프의 최소 병목 스패닝 트리(MBST)는 다음과 같이 정의된다.</p>

<ul>
<li>주어진 그래프의 모든 정점을 연결하는 부분 그래프 중에서 간선 가중치의 <strong>최댓값</strong>이 최소인 트리</li>
</ul>

<p>입력으로 주어진 그래프에 대해서, MST이면서 MBST가 아닌 것과, MBST이면서 MST가 아닌 것을 각각 찾아라.</p>

### 입력 

 <p>첫째 줄에 정수 $N,M$이 공백으로 구분되어 주어진다.</p>

<p>다음 $M$개의 줄에 그래프의 간선을 나타내는 정수 $u_i,v_i,w_i$가 공백으로 구분되어 주어진다. $i$번 간선이 정점 $u_i$와 $v_i$를 $w_i$의 가중치로 이음을 나타낸다.</p>

<p>주어진 그래프는 연결 그래프임이 보장된다.</p>

### 출력 

 <p>첫째 줄에 MST이면서 MBST가 아닌 것이 있으면 <code>YES</code>, 없으면 <code>NO</code>를 출력한다.</p>

<p><code>YES</code>를 출력한 경우 다음 $N-1$개의 줄에 MST를 이루는 간선의 번호를 출력한다.</p>

<p>다음 줄에 MBST이면서 MST가 아닌 것이 있으면 <code>YES</code>, 없으면 <code>NO</code>를 출력한다.</p>

<p><code>YES</code>를 출력한 경우 다음 $N-1$개의 줄에 MBST를 이루는 간선의 번호를 출력한다.</p>

<p>만약 정답이 여러 가지인 경우, 아무거나 하나 출력한다.</p>
